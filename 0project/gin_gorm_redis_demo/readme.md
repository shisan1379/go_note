   
# 依赖

1. 导入 `gorm` 和 `mysql` 驱动 

    ```BASH
    go get -u gorm.io/gorm  
    go get -u gorm.io/driver/mysql
   ```
    
2. 导入 `logrus` 日志包
   ```bash
   go get -u github.com/sirupsen/logrus
   ```
3. 导入 `gin` 
   
   ```bash
   go get -u github.com/gin-gonic/gin
   ```
1. 导入 `session`
   
   ```bash
   go get -u github.com/gin-contrib/sessions
   ```
