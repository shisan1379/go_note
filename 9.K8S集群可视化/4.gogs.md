# 使用docker-compose安装


拉取
```bash
docker pull gogs/gogs
```

简化搭建的流程，课程这里直接采用DockerCompose的方式搭建。docker-compose.yaml如下:
```yaml
version: "3"
volumes:
  gogsdata:
services:
  gogs:
    image: gogs/gogs
    container_name: gogs
    volumes:
      - gogsdata: /data
    restart: always
    ports:
      - "10022:22"
      - "10880:3000"
```

启动 Gogs 
```bash
docker-compose up -d
```
查看启动状态
